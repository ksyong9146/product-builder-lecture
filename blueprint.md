# Animal Face Test

## Overview

A high-quality, content-rich website designed to meet Google AdSense standards. The site provides a fun, AI-powered "Animal Face Test" that determines if a person's face is more "dog-like" or "cat-like".

## Features

*   **High-Quality Content:**
    *   **Original, Engaging Content:** Includes detailed descriptions for results, an explanation of the AI technology, and a clear site mission.
    *   **Trustworthiness (E-A-T):** Establishes expertise, authoritativeness, and trustworthiness through dedicated "About" and "Privacy Policy" pages.
*   **Professional Design:**
    *   **Multi-Page Structure:** A professional, multi-page layout with clear navigation (Home, About, Privacy).
    *   **Responsive Design:** A clean, mobile-friendly interface that provides a great user experience.
*   **Core Functionality:**
    *   **AI-Powered Test:** Utilizes a Teachable Machine model to classify user-uploaded images.
    *   **Client-Side Processing:** Ensures user privacy by processing all images directly in the browser. No data is stored or transmitted.
*   **Monetization:**
    *   **Google AdSense Integration:** Includes Google AdSense scripts and ad units, adhering to best practices for placement.

## Current Plan

The website has been significantly overhauled to meet AdSense quality guidelines. The next step is to deploy these changes.
